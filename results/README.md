# Results Directory

This directory stores the tracked benchmark evidence used by the public README
and technical documentation.

## Layout

- `smoke/`: lightweight CI-oriented benchmark outputs
- `full/`: full benchmark outputs used for public summary tables

Each suite stores:

- `runs.csv`: one row per dataset / seed / method run
- `summary.csv`: aggregated metrics by dataset and method
- `summary.md`: Markdown rendering of the aggregate summary
- `dataset_manifest.json`: manifest snapshot used when the suite was run
- `environment.json`: runtime provenance snapshot for the run environment

Optional inferential outputs:

- `summary_inference.csv`: aggregate metrics with confidence intervals and paired baseline comparisons
- `summary_inference.md`: Markdown rendering of inferential summaries

These files are generated by:

```bash
python -m ifsn_logistic.benchmarks.run --suite smoke
python -m ifsn_logistic.benchmarks.run --suite full
```
